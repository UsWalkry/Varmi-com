# Project Summary
The "Var mı?" project is a reverse marketplace platform designed specifically for the Turkish market. It allows buyers to post their needs while sellers can respond with offers. The platform incorporates a secure escrow payment system to facilitate transactions and improve user communication. Recent enhancements have concentrated on optimizing user experience, fixing performance issues, and refining core functionalities.

# Project Module Description
The application consists of several functional modules:
- **User Management**: Manages user registration, login, and profile settings.
- **Listing Management**: Facilitates the creation, viewing, and management of listings.
- **Offer Management**: Allows sellers to submit offers and buyers to review them.
- **Transaction Management**: Handles payment processing and escrow services.
- **Messaging System**: Enables real-time communication between users.
- **Favorites Management**: Lets users manage their favorite listings.
- **Admin Panel**: Provides moderation and management tools for administrators.
- **Dashboard**: Displays user statistics and interactions with listings, offers, and messages.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Node.js, NestJS (or alternatives like FastAPI/Go)
- **Database**: PostgreSQL (with Prisma ORM)
- **Deployment**: Docker, Vite for bundling, and GitHub Actions for CI/CD

# Usage
To set up the project, follow these steps:
1. Install dependencies:
   ```
   pnpm install
   ```
2. Run lint checks:
   ```
   pnpm run lint
   ```
3. Build the application:
   ```
   pnpm run build
   ```
4. Start the development server:
   ```
   pnpm run dev
   ```
