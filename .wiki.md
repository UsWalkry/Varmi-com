# Project Summary
The project "Var mı?" is a reverse marketplace platform that connects buyers and sellers, enabling buyers to post their needs while sellers respond with offers. It incorporates a secure escrow payment system to streamline transactions and improve user communication, primarily serving the Turkish market with multi-language support. Recent enhancements have focused on optimizing the messaging system and favorites management, significantly improving user experience and resolving critical rendering issues.

# Project Module Description
The application includes several functional modules:
- **User Management**: User registration, login, and profile management.
- **Listing Management**: Creation, viewing, and management of listings.
- **Offer Management**: Sellers can submit offers for listings and buyers can review them.
- **Transaction Management**: Payment processing and escrow services.
- **Messaging System**: Real-time communication between users, now with improved error handling.
- **Favorites Management**: Users can manage their favorite listings.
- **Admin Panel**: Tools for moderation and management by administrators.
- **Dashboard**: Displays user statistics and interactions with listings, offers, and messages.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Node.js, NestJS (or alternatives like FastAPI/Go)
- **Database**: PostgreSQL (with Prisma ORM)
- **Deployment**: Docker, Vite for bundling, and GitHub Actions for CI/CD

# Usage
To set up the project, follow these steps:
1. Install dependencies:
   ```
   pnpm install
   ```
2. Run lint checks:
   ```
   pnpm run lint
   ```
3. Build the application:
   ```
   pnpm run build
   ```
4. Start the development server:
   ```
   pnpm run dev
   ```
